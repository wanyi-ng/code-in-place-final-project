# Code in Place Final Project Submission - _Console "Rock, Paper, Scissor"_

Submitted by: Wanyi

Description: A simple Console Project written in Python where the user plays "Rock, Paper, Scissor". The console prints an input requesting the user to choose one of the three options; the computer randomly plays one of the options. The console prints if user or computer wins or loses, or if there was a draw.
